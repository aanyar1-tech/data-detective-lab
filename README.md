# data-detective-lab
A small Python project that analyzes a CSV of tweets — cleaning the data, finding the most-liked tweet, sorting by engagement, and searching by keyword — all without using sort(), sorted(), or max(). Includes a shell script that reports the top 5 most active users.

Files: data-detective.py (main script), top_users.sh (top users report), sample_tweets.csv (sample data).

CSV format: User,Tweet Text,Likes,Retweets
