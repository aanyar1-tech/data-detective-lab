# Lab 2: The Social Media Data Detective

## Files

- data-detective.py
- feed-analyzer.sh
- twitter_dataset.csv

## How to Run

Run the Python program:

```bash
python3 data-detective.py
```

Run the Bash script:

```bash
chmod +x feed-analyzer.sh
./feed-analyzer.sh
```

## Sorting Algorithm

This project uses the Bubble Sort algorithm to arrange tweets from the highest number of likes to the lowest. Bubble Sort repeatedly compares adjacent tweets and swaps them whenever they are in the wrong order until the list is completely sorted.
